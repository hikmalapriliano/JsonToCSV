# 📊 JSON to CSV Converter

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
![Status: Production](https://img.shields.io/badge/Status-Live-green)

**JSON to CSV Converter** adalah alat berbasis web yang efisien untuk mengubah data format JSON menjadi file CSV secara instan. Dirancang untuk kemudahan penggunaan dengan fokus penuh pada privasi data pengguna.

---

### ✨ Fitur Unggulan

* **⚡ Konversi Instan**: Cukup tempel data JSON dan hasil CSV langsung siap.
* **🔒 Privasi Terjamin**: Semua proses dilakukan di browser Anda (*client-side*). Data tidak pernah dikirim ke server mana pun.
* **💾 Download Sekali Klik**: Unduh hasil konversi langsung ke format `.csv` yang bisa dibuka di Excel atau Spreadsheet.
* **🎨 Desain Modern**: Antarmuka bersih, minimalis, dan sepenuhnya responsif untuk semua ukuran layar.
* **📝 Error Handling**: Memberikan peringatan jika format JSON yang dimasukkan tidak valid.

---

### 🛠️ Teknologi yang Digunakan

* **HTML5 & CSS3**: Struktur web modern dengan utilitas dari **Tailwind CSS**.
* **Vanilla JavaScript**: Logika *parsing* data dan algoritma konversi yang cepat.
* **Google Fonts**: Menggunakan font yang nyaman dibaca untuk pengolahan data.

---


### 🚀 Cara Menjalankan Secara Lokal
Clone repositori ini:
```bash
git clone [https://github.com/hikmalapriliano/json-to-csv.git](https://github.com/hikmalapriliano/json-to-csv.git)
 ```
Buka file `index.html` langsung di browser Anda.

Pastikan Anda terhubung ke internet untuk memuat Tailwind CSS via CDN.
